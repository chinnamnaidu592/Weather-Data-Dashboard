# 🌦 Weather Data Dashboard (Power BI)

## 📊 Project Overview
This project presents an interactive Power BI dashboard built using a weather dataset to analyze temperature, humidity, wind speed, and atmospheric pressure trends over time. The dashboard helps in understanding seasonal patterns and variations through visual analytics.

---

## 🚀 Features
- Monthly analysis of temperature, humidity, and wind speed  
- Quarterly comparison of humidity and temperature variations  
- Yearly analysis of atmospheric pressure trends  
- Interactive filters for dynamic data exploration  
- Clean and user-friendly dashboard layout  

---

## 🛠 Tools Used
- Power BI  
- CSV Dataset  
- Data Cleaning & Visualization  

---

## 📈 Key Insights
- Temperature shows a clear **seasonal trend**, peaking during mid-year months  
- Humidity levels are higher at the beginning and end of the year  
- Wind speed fluctuates across months with noticeable peaks  
- Atmospheric pressure remains relatively stable with minor yearly variations  

---


## 📌 Conclusion
This project demonstrates how Power BI can be used to transform raw weather data into meaningful insights through interactive visualizations. It highlights the importance of data storytelling in understanding real-world patterns.
